== README
{<img src="https://travis-ci.org/faissaloo/OneBoard.svg?branch=master" alt="Build Status" />}[https://travis-ci.org/faissaloo/OneBoard]  

OneBoard is an anonymous javascriptless textboard written in Ruby on Rails.  
Setup is easy:
<tt>bundle install; rake db:create db:migrate</tt>  

To start:
<tt>bin/rails server</tt>  

This is a fork of faissaloo's original code. Please check out their repo at https://github.com/faissaloo/Ratwires
